ASC X12 004010

679 Ship/Delivery Pattern Time Code
TYPE=ID MIN=1 MAX=1

Code which specifies the time for routine shipments or deliveries 


A        1st Shift (Normal Working Hours)
B        2nd Shift
C        3rd Shift
D        A.M.
E        P.M.
F        As Directed
G        Any Shift
Y        None (Also Used to Cancel or Override a Previous Pattern)
Z        Mutually Defined
