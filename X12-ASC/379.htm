ASC X12 004010

379 Bid Type Response Code
TYPE=ID MIN=2 MAX=2

Code indicating the type of response to the request for bid or quote 


AB       Alternate Bid
BF       Best and Final
BI       Bid Without Exception
BW       Bid With Exception (Request for Information)
DQ       Decline to Quote
OM       One of Multiple
RE       Replacement
RT       Request for Time Extension
UQ       Unable to Quote
