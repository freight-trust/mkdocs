ASC X12 004010

888 Nonconformance Determination Code
TYPE=ID MIN=1 MAX=1

Code indicating whether a nonconforming condition was the reason for submitting or the basis upon which a nonconformance report is initiated 


1        Order
2        Shipment
3        Billing
4        Receipt
5        Discrepant
N        Was Not a Reason for the Nonconformance Report
Y        Was a Reason for the Nonconformance Report
